## About
* Pulls 10 todos from https://jsonplaceholder.typicode.com/todos
* Adds ToDo, "pushes" to https://jsonplaceholder.typicode.com/todos
* Deletes ToDo, "deletes" from https://jsonplaceholder.typicode.com/todos
* Marks as "completed"
